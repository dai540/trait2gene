# CLI

Available commands:

- `trait2gene init CONFIG`
- `trait2gene validate CONFIG`
- `trait2gene plan CONFIG`
- `trait2gene run CONFIG`
- `trait2gene doctor [--config CONFIG]`

## Command intent

- `init` writes a minimal config template
- `validate` enforces config coherence
- `plan` shows the stage order derived from your config
- `run` writes a compact output skeleton
- `doctor` reports Python and optional config readiness
